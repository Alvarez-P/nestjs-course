## First Steps

- Install dependencies

```bash
# nestjs cli
$ npm i -g @nestjs/cli

# docker
$ npm i -g @nestjs/cli
```

- Create project

```bash
$ nest new {project-name}
```

## Project commands

- Running app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

- Running tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

- Docker

```bash
# init database
$ docker

# stop database
$ docker
```
